# 🌟 Suman Das - Personal Portfolio

A modern, responsive personal portfolio website showcasing my projects, skills, and experience in AI & Machine Learning development.

## 🎨 Features

- **Modern Dark Theme** - Beautiful gradient backgrounds and glassmorphism effects
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Fade-in effects, hover animations, and smooth scrolling
- **Interactive Elements** - Hamburger menu, active link highlighting, form handling
- **Project Showcase** - 13+ projects with detailed descriptions and tech stacks
- **PDF Resume** - Download professional resume with one click
- **Contact Form** - Fully functional contact form with validation

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/suman6532/my-protfolio.git
   cd my-protfolio
   ```

2. **Open in browser**
   - Double-click `index.html` or
   - Use a local server: `python -m http.server 8000`

3. **Customize**
   - Edit `index.html` to add your content
   - Modify colors in `style.css` (CSS variables at top)
   - Update projects and skills sections

## 📝 Customization Guide

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #00d4ff;
    --secondary-color: #ff006e;
    --accent-color: #8338ec;
    --dark-bg: #0a0e27;
}
```

### Update Personal Info
- Edit contact details in HTML
- Update project descriptions
- Modify skills in the skills section

### Add New Sections
- Follow the existing structure
- Copy a section and modify
- Add fade-in class for animations
